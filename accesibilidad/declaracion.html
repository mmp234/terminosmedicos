Declaración de Accesibilidad

Situación de cumplimiento

Aprendizaje sobre términos médicos se compromete a proporcionar un sitio web accesible para todos los usuarios, independientemente de su discapacidad. Nos esforzamos por cumplir con las WCAG 2.1, publicadas por, en inglés, el W3C, siendo nuestro sitio web parcialmente conforme con el nivel AA.

Medidas de Accesibilidad Implementadas

  • Identificación del idioma
  • Etiquetado semántico
  • Texto alternativo para imágenes
  • Mecanismos de navegación
  • Enlaces
  • Formularios
  • Títulos

Identificación del idioma

El idioma principal de esta página web es el español, indicado mediante el atributo lang="es" en la etiqueta <html>. Los cambios de idioma dentro del contenido se identifican utilizando el atributo lang en las etiquetas correspondientes. (WCAG 2.1, Criterio 3.1.1 - Nivel A, Criterio 3.1.2 - Nivel AA)

Etiquetado semántico

Utilizamos etiquetas semánticas de HTML5 para estructurar el contenido de la página de forma significativa, incluyendo <header>, <nav>, <main>, <article>, <section>, <footer>, etc. Las etiquetas de encabezado (<h1> a <h6>) se utilizan para estructurar la página con títulos y subtítulos, siguiendo un orden jerárquico. (WCAG 2.1, Criterio 1.3.1 - Nivel A)

Texto alternativo para imágenes

Todas las imágenes de este sitio web tienen un texto alternativo (atributo alt) que describe su función. Las imágenes decorativas tienen un atributo alt vacío (alt=""). (WCAG 2.1, Criterio 1.1.1 - Nivel A)

Mecanismos de navegación

Este sitio web proporciona un mapa de navegación y una barra de posición al inicio de cada página para facilitar la navegación. Los textos de los enlaces son claros y descriptivos, lo que permite a los usuarios comprender su propósito. (WCAG 2.1, Criterio 2.4.5 - Nivel AA, Criterio 2.4.8 - Nivel AAA)

Enlaces

El texto de los enlaces es significativo, descriptivo y comprensible fuera de contexto. Se evita el uso de "clic aquí" o frases genéricas. Cada enlace tiene un texto único. (WCAG 2.1, Criterio 2.4.4 - Nivel A)

Formularios

Las instrucciones del formulario son claras y concisas, se indican los campos obligatorios con un asterisco y se explican los formatos de entrada.

Se utiliza label con el atributo for para asociar etiquetas a controles.

Se agrupan controles relacionados con fieldset y lengend, especialmente botones de opción y casillas de verificación.

Se verifica el orden de tabulación para asegurar una navegación lógica con el teclado.

(WCAG 2.1, Criterio 3.3.2 - Nivel A)

Títulos

Cada página tiene un título único y descriptivo definido con la etiqueta title, incluyendo el título de la página y el título del sitio web. (WCAG 2.1, Criterio 2.4.2 - Nivel A)

Conformidad

Esta declaración de accesibilidad fue actualizada por última vez el 10/10/2024. Si bien nos esforzamos por hacer que nuestro sitio web sea accesible para todos, reconocemos que puede haber algunas limitaciones. Si experimentas algún problema de accesibilidad, por favor, ponte en contacto con nosotros a través del formulario de contacto para que podamos ayudarte.
